# Use Taskwarrior 3.x with TaskChampion sync on atlas

Taskwarrior 3.x is the single source of truth for personal task management, syncing through an official TaskChampion sync server (SQLite, Docker) on atlas behind the existing Traefik + Private Route53 + Tailscale pattern. The `task` CLI cannot present OIDC bearer tokens, so the sync server is an infra-surface secured by Tailscale ACLs + a client-ID allowlist + client-side AEAD encryption, not Authentik. iOS uses Taskchamp (native TC replica); Mac uses `taskwarrior-tui` for terminal triage and a LaunchAgent for periodic sync; atlas runs Sisyphus as a write-capable Kanban projection backed by its own Taskwarrior replica and protected at Traefik by Authentik forward-auth. The deployment/service identifier remains `taskboard`. Timewarrior is removed entirely — recurring tasks are native Taskwarrior and time tracking had no sync story.
